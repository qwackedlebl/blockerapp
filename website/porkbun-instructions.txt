==================================================
PORKBUN HOSTING SETUP INSTRUCTIONS FOR ROTLOCKER
==================================================

IMPORTANT: Porkbun's HTML editor doesn't support file uploads directly.
You need to host the APK file externally.

OPTION 1: Use GitHub (Recommended - Free)
==========================================

1. Create a GitHub account (if you don't have one)
2. Create a new repository called "rotlocker"
3. Upload rotlocker.apk to the repository
4. Go to Settings → Pages → Enable GitHub Pages
5. Get the direct link to your APK:
   https://[your-username].github.io/rotlocker/rotlocker.apk
6. Use this link in the HTML below (replace the APK_DOWNLOAD_LINK)

OPTION 2: Use Google Drive
===========================

1. Upload rotlocker.apk to Google Drive
2. Right-click → Share → Change to "Anyone with the link"
3. Copy the sharing link
4. Convert it to direct download:
   - Original: https://drive.google.com/file/d/FILE_ID/view?usp=sharing
   - Direct: https://drive.google.com/uc?export=download&id=FILE_ID
5. Use this link in the HTML below

OPTION 3: Use Dropbox
======================

1. Upload rotlocker.apk to Dropbox
2. Get sharing link
3. Change the end from ?dl=0 to ?dl=1
4. Use this link in the HTML below

PORKBUN SETUP STEPS:
====================

1. Log into your Porkbun account
2. Go to your domain management
3. Click "Website Builder" or "HTML Editor"
4. Delete all existing content
5. Copy the HTML from 'porkbun.html' file
6. Replace "APK_DOWNLOAD_LINK" with your actual APK link from above
7. Click Save/Publish

Your website will be live at your domain!
